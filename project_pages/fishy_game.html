Fishy Game

Fishy Game is an endless runner with a fishy protagonist. Jump and crouch to avoid evil seaweed and enemy fish as you run. The game gets progressively faster as our fishy hero avoids more enemies. There are power ups that can help, or hinder your gameplay. Perhaps you’ll become invincible, or you’ll end up being slowed down. Or maybe you’ll jump twice as high. What you get is up to chance. You have three attempts to achieve a high score. Will you swim to the top, or will you sleep with the fishes?

About

Fishy Run is an endless runner inspired by Google's Dino Run. Fishy Run was made with the Dragonfly game engine.

My Role

The design and implementation of this project was shared between myself and my teamate. My personal contributions included implementation of the following:

  • endless run functionality, jump mechanic, and enemies.
  • powerup system.
  • scoring and life system.
